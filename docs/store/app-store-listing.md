# App Store 掲載情報 (#51)

App Store Connect に登録する日本語メタデータ。値の実体は
[`fastlane/metadata/ja/`](../../fastlane/metadata/ja/) に配置し、#55 の fastlane deliver で
そのまま同期できる構成にしている。

## フィールド一覧

| フィールド | ファイル | 上限 | 内容 |
| --- | --- | --- | --- |
| App 名 | `name.txt` | 30 | サブミル |
| サブタイトル | `subtitle.txt` | 30 | 学生のサブスク見直しアプリ |
| キーワード | `keywords.txt` | 100 | サブスク管理・節約・学割 等(カンマ区切り) |
| プロモーションテキスト | `promotional_text.txt` | 170 | 見直し訴求のフック(審査なしで随時更新可) |
| 概要(Description) | `description.txt` | 4000 | ペイン訴求 + 機能列挙 + 学割アピール |
| リリースノート | `release_notes.txt` | 4000 | v1.0 初回リリース用 |
| プライバシーポリシー URL | `privacy_url.txt` | — | GitHub Pages (#52) |

> 文字数は全角も 1 文字としてカウント(App Store の仕様)。各ファイルは末尾改行なしの
> プレーンテキスト。

## 提出前に確定が必要な項目 (TODO)

- [ ] **サポート URL(必須)**: App Store は新規アプリにサポート URL を必須とする。
      問い合わせ先や FAQ を載せた実在ページが必要。未整備のため `support_url.txt` は
      あえて未作成。専用ページ(例: `submil-content` に `support.html` を追加)を用意して設定する。
- [ ] **マーケティング URL(任意)**: ランディングページがあれば `marketing_url.txt` を追加。
- [ ] カテゴリ: 主「ファイナンス」/ 副「ユーティリティ」を想定(ASC 側で設定、#54)。
- [ ] 年齢制限レーティング: ASC のアンケートで確定(#54)。
- [ ] 連絡先メール等は #52/#53 と同じ暫定値(`hikaruken0126@gmail.com`)。専用アドレスにするなら合わせて差し替え。

## 補足

- 概要は「悩み → 機能 → 学割 → 安心」の順。広告表示・PR 表記の注意書きを末尾に記載
  (景表法・ステマ規制および実装 (#44) と整合)。
- キーワードはスペースを入れず純カンマ区切り(スペースは文字数を消費するため)。
- 英語(en-US)ロケールを追加する場合は `fastlane/metadata/en-US/` に同名ファイルを置く。
